# Raykat
Simple raytracing engine in C (WIP).
Based on [_Ray Tracing in One Weekend_](https://raytracing.github.io/books/RayTracingInOneWeekend.html) by Peter Shirley.

Watch [image.png](https://raw.githubusercontent.com/ktkk/raykat/master/image.png) for a rough estimate of where I am through the course.

## Build
Build with make:
```sh
make
```

To generate the image, run the make rule "image":
```sh
make image
```
